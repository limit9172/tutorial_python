Pendahuluan Python

Python adalah bahasa pemrograman tingkat tinggi yang pertama kali dirilis oleh Guido van Rossum pada tahun 1991.

Keunggulan Python:

  • Mudah dipelajari dan digunakan
  • Memiliki komunitas yang besar
  • Digunakan dalam banyak bidang seperti web, AI, data science

Variabel dalam Python

Variabel adalah tempat untuk menyimpan data dalam program.

Aturan penamaan variabel dalam Python:

  • Harus dimulai dengan huruf atau garis bawah (_)
  • Tidak boleh menggunakan spasi
  • Bersifat case-sensitive (besar kecil huruf berpengaruh)

# Contoh variabel
nama = "Ali"
umur = 20
tinggi = 175.5
print(nama, umur, tinggi)

Tipe Data dalam Python

Python memiliki berbagai tipe data dasar, antara lain:

  • String (str)
  • Integer (int)
  • Float (float)
  • Boolean (bool)
  • List, Tuple, Dictionary, dan Set

# Contoh tipe data
x = "10"  # String
y = int(x)  # Konversi ke Integer
print(y + 5)

Percabangan dalam Python

Python menggunakan `if-elif-else` untuk membuat keputusan berdasarkan kondisi.


umur = 18
if umur >= 17:
    print("Kamu boleh membuat SIM")
else:
    print("Kamu belum cukup umur")

Perulangan dalam Python

Python memiliki dua jenis perulangan utama:

  • Perulangan `for` untuk mengulang dalam rentang tertentu.
  • Perulangan `while` untuk mengulang selama kondisi masih benar.

# Looping dengan for
for i in range(5):
    print("Hello, Python!")

# Looping dengan while
count = 0
while count < 5:
    print("Perulangan ke-", count)
    count += 1
